Shorten links, organize campaigns, and see who's clicking - all from your WordPress dashboard.

== About This Plugin ==

Shortlinkr turns long, messy URLs into short, clean links that are easy to share. Everything runs directly from your WordPress dashboard - no external services needed.

**What You Can Do:**

* **Shorten links**: Turn giant URLs into short, memorable links
* **Organize by campaigns**: Group your links by project, client, or campaign
* **Track results**: See how many clicks each link gets and where they come from
* **Import and export**: Move your links between sites easily
* **Works with multisite**: No problems with WordPress multisite networks
* **Custom codes**: Pick short codes that make sense to you
* **Easy to use**: Clean, simple dashboard

**Great For:**

* Marketing campaigns
* Social media posts
* Email newsletters
* Affiliate marketing
* Event tracking
* A/B testing

**Privacy & Security:**

Your data stays in your own WordPress database. Nothing gets sent to outside services.

== How to Install ==

**Easy way:**

1. Log into your WordPress dashboard
2. Go to Plugins > Add New
3. Search for "Shortlinkr"
4. Click "Install Now" then "Activate"

**Manual way:**

1. Download the plugin ZIP file
2. Log into your WordPress dashboard
3. Go to Plugins > Add New > Upload Plugin
4. Pick the ZIP file and click "Install Now"
5. After it installs, click "Activate Plugin"

**After you activate it:**

1. Look for "Shortlinkr" in your WordPress menu
2. Create your first short link
3. Set it up how you like

== Common Questions ==

= Does it work with multisite? =

Yes! Works perfectly with WordPress multisite. You can enable it network-wide or just on specific sites.

= Where does my data go? =

Everything stays in your own WordPress database. We don't use any external services.

= Can I export my links? =

Yes! There's import/export built in so you can move links between sites.

= Can I customize the short URLs? =

Yes! You pick whatever short code you want when creating each link.

= Does it show statistics? =

Yes! You can see clicks, where visitors came from, and other helpful data.

= Can I organize links into campaigns? =

Yes! Create campaigns to group and track related links together.

= What happens if I deactivate the plugin? =

Your data stays in the database, but the short links stop working. When you reactivate, everything comes back.
